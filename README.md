# SpeedUP

## 针对大多数视频网站的倍速播放插件

一般的视频网站只是提供了0.5到2倍速的视频播放，如果想
要更快的倍速播放，我之前的做法是直接在开发者工具中设置
页面video的属性。每次都要修改有点麻烦，所以就做了这样
一个插件。目前只针对bilibili进行调整，后续会添加更多

多数视频的播放页面一般也只会同时播放一个视频
